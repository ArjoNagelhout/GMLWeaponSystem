# GMLWeaponSystem
An inventory based weapon system for GameMaker:Studio 2

[YouTube demo](https://www.youtube.com/watch?v=WXPy7eFRIes&t=5s)

Features / controls:
- WASD to move around smoothly (8 directions)
- Aim with mouse
- Click to shoot
- Select weapon by pressing 1, 2, 3, 4 or 5
- E to pick up weapon
- Q to drop weapon
